# ai201331223113
一个基于 Pygame 开发的躲避类游戏，玩家需要控制角色躲避不断出现的障碍物和智能追踪的 AI 敌人，收集道具获得分数和特殊效果，挑战最高记录！

**游戏介绍**

    在 AI Dodger 中，你将控制一个彩色圆形角色，通过鼠标移动躲避从屏幕四周不断生成的障碍物。随着游戏进行，还会出现具有追踪功能的 AI 敌人，游戏难度会逐渐提升。收集各种道具可以帮助你获得更高分数或生存更久，游戏支持分数记录和排名系统，快来挑战你的极限吧！

**主要功能**

    🎨 6 种玩家颜色可选，自定义游戏体验
    
    🚀 智能 AI 追踪系统，敌人会主动追击玩家
    
    🔧 4 种实用道具：加分、额外生命、清屏炸弹、减速效果
    
    📊 完整的排名系统，记录历史最高分和游戏记录
    
    ⏸️ 游戏暂停 / 继续功能，支持随时中断和恢复
    
    🎮 直观的操作提示和游戏状态显示
    
    💻 跨平台支持（Windows、macOS、Linux）

**安装要求
环境依赖**

  Python 3.6 或更高版本
  Pygame 1.9.6 或更高版本

**安装步骤**

1、克隆本仓库到本地：

    git clone https://github.com/your-username/ai-dodger.git
    
    cd ai-dodger

2、安装依赖包：

    pip install pygame

3、运行游戏：

    python ai_dodger.py

**游戏机制说明**

角色系统

    初始生命：3 条
  
    角色大小：固定 25 像素
    
    移动速度：根据鼠标距离自适应，最大速度 8 像素 / 帧
    
障碍物系统

    生成位置：屏幕四个边缘随机位置
  
    移动方向：朝向玩家当前位置
    
    大小范围：15-30 像素随机
    
    生成速度：随着分数增加而加快（初始 30 帧生成 1 个，最低 15 帧生成 1 个）
  
AI 追踪者系统

    生成条件：分数每增加 500 分，且当前 AI 数量少于 5 个
    
    移动特性：会追踪玩家位置，带有轻微随机偏移
    
    碰撞惩罚：碰撞后扣除 2 条生命

**如果你想对游戏进行自定义修改，可以参考以下几个方向：**

    添加更多颜色：修改 PLAYER_COLORS 和 PLAYER_COLOR_NAMES 列表
    
    调整游戏难度：修改障碍物生成速度、AI 追踪速度等参数
    
    添加新道具：在 spawn_powerup 和 apply_powerup 方法中扩展
    
    修改游戏界面：调整窗口大小、字体样式、颜色搭配等
    
    添加音效：集成 Pygame 的混音功能，添加背景音乐和音效
